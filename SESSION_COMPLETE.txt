  :Continue

  - Top pane: Chat history (read-only)
  - Bottom pane: Input area (type here)

To send a message:
  1. Type in the input area
  2. Press <CR>
  3. Message sends and input clears

Keymaps:
  <CR>     - Send message (from input area)
  q or Esc - Close chat window
  i or a   - Enter insert mode (to type)
